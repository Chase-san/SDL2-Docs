SDL_AtomicSetPtr

Use this function to set a pointer to a value atomically.

Syntax

void* SDL_AtomicSetPtr(void** a,
                       void*  v)

Function Parameters

a a pointer to a pointer
v the desired pointer value

Return Value

Returns the previous value of the pointer.
