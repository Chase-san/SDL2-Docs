SDL_EnclosePoints

Use this function to calculate a minimal rectangle enclosing a set of points.

Syntax

SDL_bool SDL_EnclosePoints(const SDL_Point* points,
                           int              count,
                           const SDL_Rect*  clip,
                           SDL_Rect*        result)

Function Parameters

pointsan array of SDL_Point structures representing points to be enclosed
countthe number of structures in the points array
clipan SDL_Rect structure used for clipping or NULL to enclose all points; see Remarks for details
resultan SDL_Rect structure filled in with the minimal enclosing rectangle

Return Value

Returns SDL_TRUE if any points were enclosed or SDL_FALSE if all the points were outside of the clipping rectangle.

Code Examples

Remarks

If clip is not NULL then only points inside of the clipping rectangle are considered.
