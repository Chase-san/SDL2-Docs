SDL_GetWindowMaximumSize

Use this function to get the maximum size of a window's client area.

Syntax

void SDL_GetWindowMaximumSize(SDL_Window* window,
                              int*        w,
                              int*        h)

Function Parameters

windowthe window to query
wa pointer filled in with the maximum width of the window
ha pointer filled in with the maximum height of the window
