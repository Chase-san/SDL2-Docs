SDL_GetWindowPosition

Use this function to get the position of a window.

Syntax

void SDL_GetWindowPosition(SDL_Window* window,
                           int*        x,
                           int*        y)

Function Parameters

windowthe window to query
xa pointer filled in with the x coordinate of the window
ya pointer filled in with the y coordinate of the window
