SDL_HINT_RENDER_SCALE_QUALITY

A hint that specifies scaling quality.

Values

0 or nearestnearest pixel sampling
1 or linearlinear filtering (supported by OpenGL and Direct3D)
2 or bestanisotropic filtering (supported by Direct3D)

Default

By default nearest pixel sampling is used.

Code Examples

Remarks

This hint is checked when a texture is created and it affects scaling when copying that texture.
