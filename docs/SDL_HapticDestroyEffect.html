SDL_HapticDestroyEffect

Use this function to destroy a haptic effect on the device.

Syntax

void SDL_HapticDestroyEffect(SDL_Haptic* haptic,
                             int         effect)

Function Parameters

hapticthe SDL_Haptic device to destroy the effect on
effectID identifier of the haptic effect to destroy

Code Examples

Remarks

This will stop the effect if it's running. Effects are automatically destroyed when the device is closed.
