SDL_HapticStopEffect

Use this function to stop the haptic effect on its associated haptic device.

Syntax

int SDL_HapticStopEffect(SDL_Haptic* haptic,
                         int         effect)

Function Parameters

hapticthe SDL_Haptic device to stop the effect on
effectID identifier of the haptic effect to stop

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
