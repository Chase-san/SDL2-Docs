SDL_HasIntersection

Use this function to determine whether two rectangles intersect.

Syntax

SDL_bool SDL_HasIntersection(const SDL_Rect* A,
                             const SDL_Rect* B)

Function Parameters

Aan SDL_Rect structure representing the first rectangle
Ban SDL_Rect structure representing the second rectangle

Return Value

Returns SDL_TRUE if there is an intersection, SDL_FALSE otherwise.
