SDL_HasSSE41

Use this function to determine whether the CPU has SSE4.1 features.

Syntax

SDL_bool SDL_HasSSE41(void)

Return Value

Returns SDL_TRUE if the CPU has SSE4.1 features or SDL_FALSE if not.
