SDL_RenderDrawRect

Use this function to draw a rectangle on the current rendering target.

Syntax

int SDL_RenderDrawRect(SDL_Renderer*   renderer,
                       const SDL_Rect* rect)

Function Parameters

rendererthe rendering context
rectan SDL_Rect structure representing the rectangle to draw, or NULL to outline the entire rendering target

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
