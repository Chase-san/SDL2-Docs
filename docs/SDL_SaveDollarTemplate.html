SDL_SaveDollarTemplate

Use this function to save a currently loaded Dollar Gesture template.

Syntax

int SDL_SaveDollarTemplate(SDL_GestureID gestureId,
                           SDL_RWops*    dst)

unction Parameters

gestureIda gesture id
dsta SDL_RWops to save to

Return Value

Returns 1 on success or 0 on failure; call SDL_GetError() for more information.
