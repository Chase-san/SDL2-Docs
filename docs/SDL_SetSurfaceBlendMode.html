SDL_SetSurfaceBlendMode

Use this function to set the blend mode used for blit operations.

Syntax

int SDL_SetSurfaceBlendMode(SDL_Surface*  surface,
                            SDL_BlendMode blendMode)

Function Parameters

surfacethe SDL_Surface structure to update
blendModethe SDL_BlendMode to use for blit blending; see Remarks for details

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.

Code Examples

Remarks

blendMode may be one of the following:

SDL_BLENDMODE_NONEno blending
 dstRGBA = srcRGBA
SDL_BLENDMODE_BLENDalpha blending
 dstRGB = (srcRGB * srcA) + (dstRGB * (1-srcA))
 dstA = srcA + (dstA * (1-srcA))
SDL_BLENDMODE_ADDadditive blending
 dstRGB = (srcRGB * srcA) + dstRGB
 dstA = dstA
SDL_BLENDMODE_MODcolor modulate
 dstRGB = srcRGB * dstRGB
 dstA = dstA
