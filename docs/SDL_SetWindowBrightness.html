SDL_SetWindowBrightness

Use this function to set the brightness (gamma correction) for a window.

Syntax

int SDL_SetWindowBrightness(SDL_Window* window,
                            float       brightness)

Function Parameters

windowthe window to change
brightnessthe brightness (gamma) value to set where 0.0 is completely dark and 1.0 is normal brightness

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
