SDL_SwapBE64

Use this function to swap the byte order of a 64-bit big-endian value to native ordering.

Syntax

Uint64 SDL_SwapBE64(Uint64 x)

Function Parameters

xthe value to be swapped

Return Value

Returns the native 64-bit value.
