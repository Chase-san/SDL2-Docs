SDL_TextureModulate

An enumeration of the texture channel modulation used in SDL_RenderCopy().

Values

SDL_TEXTUREMODULATE_NONEno modulation
SDL_TEXTUREMODULATE_COLORsrcC = srcC * color
SDL_TEXTUREMODULATE_ALPHAsrcA = srcA * alpha
